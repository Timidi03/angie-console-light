## NGINX Plus Dashboard

Live activity monitoring tool for NGINX Plus.

### Build process:

Ensure you have all dependencies installed:
```
yarn install
```

To build the dashboard run:
```
yarn build
```
The result will be placed in `dist/dashboard.html`.
